# Sketch Export Options
Sketch plugin to help manage export formats
## Installation
1. Clone or download the code
2. Double click on ExportOptions.sketchplugin
3. Sketch should automatically install the plugin

### Manual Installation
The above should handle installing the plugin but failing that it can be manually copied to the plugins folder by doing the following:

1. Open Sketch
1. In the menu bar select the Plugins > Manage Plugins...
1. Click the gear icon in the bottom left or right-click the list area
1. Click Show Plugins Folder
1. Copy ExportOptions.sketchplugin to the folder displayed

## Usage
The plugin allows for adding or remove export formats. To do so, do the following:

1. Select the layers to add/remove the export formats to/from
1. In the menu bar select Plugins > Export Options
1. Select the action that you would like to perform (add/remove remove formats)

Alternatively the following shortcut keys are provied
- Control + Shift + R = Remove all export sizes
- Control + Shift + E = Add Android export sizes
